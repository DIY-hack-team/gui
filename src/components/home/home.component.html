Исходный код

Проект состоит из 3-х репозиториев:

  1. gui: https://github.com/DIY-hack-team/gui
  2. backend: https://github.com/DIY-hack-team/backend
  3. infra: https://github.com/DIY-hack-team/infra

Проект gui написан на фреймворке angular. Настроена непрерываная сборка - скрипт deploy.sh. Его можно интегрировать в любую CI/CD.
Проект backend написан на фреймворке nestjs. Настроена непрерываная сборка - скрипт deploy.sh. Его можно интегрировать в любую CI/CD.
Проект infra хранит скрипты настройки PostgreSQL и nginx. Конфигурацие обновляется с помощью скрипта deploy.sh
Все секреты из проекта убраны. Написаны заглушки для настроек в файле .env.local.env. Настройки prod backend находятся в .env.prod.env
